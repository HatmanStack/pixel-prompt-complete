# Sound Effects

This directory contains sound effect files for user interactions.

## Required Sound Files

Place the following sound files in this directory:

- `click.wav` or `click.mp3` - Button click sound
- `switch.wav` or `switch.mp3` - Toggle/slider change sound
- `swoosh.wav` or `swoosh.mp3` - Transition sound
- `expand.mp3` or `expand.wav` - Expand/collapse section sound

## Sound File Requirements

- Format: WAV or MP3
- Size: < 50KB per file (keep them small)
- Duration: < 1 second (short, punchy sounds)
- Volume: Normalized to prevent clipping

## Sources for Free Sound Effects

- **Pixabay**: https://pixabay.com/sound-effects/
- **Freesound**: https://freesound.org/
- **Zapsplat**: https://www.zapsplat.com/
